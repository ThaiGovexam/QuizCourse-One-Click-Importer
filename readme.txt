นำเข้าคอร์สเรียนและแบบทดสอบจากไฟล์ Excel/CSV เพียงครั้งเดียว สำหรับผู้ประกอบการออนไลน์ที่ต้องการประหยัดเวลา

== Description ==

**QuizCourse One-Click Importer** ช่วยให้คุณสร้างคอร์สเรียนและแบบทดสอบได้อย่างรวดเร็วด้วยการนำเข้าข้อมูลครั้งเดียว เหมาะสำหรับผู้ประกอบการออนไลน์ นักการตลาดดิจิทัล และผู้สร้างคอนเทนต์ที่ต้องการสร้างคอร์สเรียนออนไลน์จำนวนมากในเวลาอันรวดเร็ว

### ปัญหาที่ปลั๊กอินนี้แก้ไข

ในระบบปกติ คุณต้องทำงานหลายขั้นตอน:
1. อัพโหลดคำถาม (Questions)
2. สร้างแบบทดสอบ (Quizzes)
3. รวมแบบทดสอบเป็นคอร์ส (Courses)

แต่ด้วย **QuizCourse One-Click Importer** คุณสามารถทำทั้งหมดได้ในขั้นตอนเดียว!

### คุณสมบัติหลัก

* **นำเข้าข้อมูลแบบครบวงจร** - สร้างคอร์ส, เซคชั่น, ควิซ, คำถาม และคำตอบในครั้งเดียว
* **รองรับไฟล์หลายรูปแบบ** - Excel (.xlsx, .xls) และ CSV
* **ตัวช่วยจับคู่ฟิลด์ (Field Mapping)** - กำหนดความสัมพันธ์ระหว่างคอลัมน์ในไฟล์กับฟิลด์ในฐานข้อมูล
* **ตรวจสอบข้อมูลอัตโนมัติ** - ตรวจสอบความถูกต้องของข้อมูลก่อนนำเข้า
* **ดาวน์โหลดเทมเพลต** - มีตัวอย่างไฟล์ให้ดาวน์โหลดเพื่อความสะดวกในการเตรียมข้อมูล
* **แสดงความคืบหน้า** - ติดตามสถานะการนำเข้าข้อมูลแบบเรียลไทม์
* **รายงานผลโดยละเอียด** - ดูสรุปจำนวนข้อมูลที่นำเข้าสำเร็จ

== Installation ==

1. อัพโหลดโฟลเดอร์ `quizcourse-importer` ไปยังไดเรกทอรี `/wp-content/plugins/` หรือติดตั้งผ่านหน้า Plugins ใน WordPress โดยตรง
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress
3. เข้าถึงการตั้งค่าปลั๊กอินผ่านเมนู 'QuizCourse Import' ที่ปรากฏในแดชบอร์ด
4. ดาวน์โหลดไฟล์เทมเพลตตัวอย่างเพื่อเตรียมข้อมูลของคุณ
5. อัพโหลดไฟล์ Excel/CSV ที่มีข้อมูลของคุณตามรูปแบบที่กำหนด

== Frequently Asked Questions ==

= ปลั๊กอินนี้รองรับ LMS Plugin ตัวไหนบ้าง? =

ปลั๊กอินนี้ออกแบบมาให้ทำงานร่วมกับระบบ LMS ที่ใช้งานแพร่หลายใน WordPress เช่น LearnDash, Tutor LMS, LearnPress และระบบ Quiz อื่นๆ ที่มีโครงสร้างฐานข้อมูลมาตรฐาน

= ฉันสามารถดูตัวอย่างไฟล์นำเข้าได้หรือไม่? =

ได้! ภายในปลั๊กอินมีปุ่มให้ดาวน์โหลดไฟล์เทมเพลตตัวอย่าง ซึ่งจะช่วยให้คุณเข้าใจถึงโครงสร้างข้อมูลที่จำเป็นสำหรับการนำเข้า

= ปลั๊กอินนี้รองรับการนำเข้ารูปภาพหรือไม่? =

ใช่ คุณสามารถระบุ URL ของรูปภาพในไฟล์นำเข้าได้ โดยปลั๊กอินจะดึงรูปภาพนั้นมาใช้กับคอร์สเรียน, แบบทดสอบ, หรือคำถามของคุณ

= ฉันสามารถกำหนดการแมป (mapping) ฟิลด์เองได้หรือไม่? =

ได้ ปลั๊กอินมีระบบ Field Mapping ที่ยืดหยุ่น ช่วยให้คุณสามารถกำหนดความสัมพันธ์ระหว่างคอลัมน์ในไฟล์ของคุณกับฟิลด์ในฐานข้อมูล WordPress ได้

= หากเกิดข้อผิดพลาดระหว่างการนำเข้า จะเกิดอะไรขึ้น? =

ระบบใช้ Transaction ในการนำเข้าข้อมูล หมายความว่าหากเกิดข้อผิดพลาดระหว่างการนำเข้า ระบบจะ Rollback ข้อมูลทั้งหมด เพื่อป้องกันไม่ให้มีข้อมูลที่ไม่สมบูรณ์ในระบบของคุณ

= ฉันสามารถนำเข้าคำถามจำนวนมากได้หรือไม่? =

ได้ ปลั๊กอินออกแบบมาให้รองรับการนำเข้าข้อมูลจำนวนมาก อย่างไรก็ตามประสิทธิภาพอาจขึ้นอยู่กับการตั้งค่าเซิร์ฟเวอร์ของคุณ

= มีรูปแบบคำถามใดบ้างที่รองรับ? =

ขึ้นอยู่กับระบบ LMS ที่คุณใช้ แต่โดยทั่วไปแล้วปลั๊กอินรองรับคำถามหลายรูปแบบ เช่น ปรนัย, ถูก/ผิด, เรียงลำดับ, จับคู่, และอื่นๆ

== คู่มือการใช้งานฉบับย่อ ==

1. **เตรียมข้อมูล**
   - ดาวน์โหลดไฟล์เทมเพลตตัวอย่าง
   - กรอกข้อมูลคอร์ส, เซคชั่น, ควิซ, คำถาม และคำตอบลงในชีตที่เหมาะสม
   - บันทึกไฟล์ในรูปแบบ Excel หรือ CSV

2. **อัพโหลดไฟล์**
   - ไปที่เมนู QuizCourse Import
   - อัพโหลดไฟล์ Excel/CSV ของคุณ
   - รอให้ระบบตรวจสอบข้อมูลเบื้องต้น

3. **แมปฟิลด์**
   - ตรวจสอบและปรับแต่งการจับคู่ฟิลด์ในไฟล์กับฐานข้อมูล
   - คลิกปุ่ม "เริ่มนำเข้าข้อมูล"

4. **รอการประมวลผล**
   - ระบบจะแสดงความคืบหน้าในการนำเข้าข้อมูล
   - เมื่อเสร็จสิ้น คุณจะเห็นรายงานสรุปผลการนำเข้า

5. **ตรวจสอบข้อมูล**
   - เข้าถึงคอร์สและแบบทดสอบที่สร้างขึ้นผ่านเมนูที่เกี่ยวข้อง
   - ตรวจสอบความถูกต้องและปรับแต่งเพิ่มเติมหากจำเป็น

== ข้อกำหนดด้านเทคนิค ==

* WordPress 5.0 หรือสูงกว่า

* การเปิดใช้งาน PHP extensions: zip, xml, mbstring
